## Числа Фибоначчи
Программа выводит числа Фибоначчи не больее 186, т.к. на 187 числе uint128 переполняется.

Идея алгоритма в том, что 

$$
\begin{pmatrix}
1 & 1 \\
1 & 0
\end{pmatrix}^n = \begin{pmatrix}
F_{n+1} & F_n \\
F_n & F_{n-1}
\end{pmatrix}
$$

плюс алгоритм быстрого возведения в степень дает время вычисления чисел Фибоначчи меньшее, чем $O(n)$, и даже близкое к $O(\log n)$.

## Запуск
* Программа
  ```bash
  $ c3c run
  ```
* Тесты
  ```bash
  $ c3c test
  ```

```
|  |--test_fib.c3  -- модуль с тестами для функций из fib
```
